# fleabag-quotes
Fleabag quotes API

## Pre-requisites
- Node (v18.17.0)
- NPM (v9.6.7)

## Installation
```bash
$ npm install
```

## Test
```bash
$ npm test
```

## Run
```bash
$ npm start
> fleabag-quotes@1.0.0 start
> ts-node src/server.ts

Server running at http://localhost:3000
```
